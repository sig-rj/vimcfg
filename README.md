# Vim

Edit vimrc

```
:edit $MYVIMRC
```

Testar as modificações

```
:source $MYVIMRC
```

# bundles

## VundleVim/Vundle.vim
Utilizado para a instalação dos plugins, instruções de instalação em:

[https://github.com/VundleVim/Vundle.vim]

Para Instalar/reinstalar plugins

```
:PluginInstall
```
